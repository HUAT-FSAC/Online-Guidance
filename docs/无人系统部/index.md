# 无人系统部

## 简介

无人系统部分由基于相机的障碍物检测系统、基于激光雷达的障碍物检测系统、基于相机与激光雷达融合的障碍物检测系统、组合导航系统、路径规划系统、车辆纵横向控制系统以及仿真平台系统组成。
